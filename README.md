## Описание программы

Weather Diary - это графическое приложение для ведения дневника погоды. Позволяет добавлять, просматривать, фильтровать и удалять записи о погоде. Данные сохраняются в JSON-файл и автоматически загружаются при запуске.

### Функциональность

-  Добавление записей с датой, температурой, описанием и осадками
-  Просмотр всех записей в виде таблицы
-  Фильтрация по дате (показать записи за определённый день)
-  Фильтрация по температуре (показать записи выше заданного значения)
-  Удаление выбранных записей
-  Сохранение и загрузка данных из JSON-файла
-  Валидация ввода (формат даты, числовое значение температуры, непустое описание)
